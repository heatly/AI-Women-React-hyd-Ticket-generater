# AI Conclave: Women's Edition - Ticket Generator

A premium, frontend-only React application to generate personalized event tickets with AI-themed aesthetics.

## 🚀 Features
- **Pure Frontend Generation**: Uses HTML5 Canvas for 100% mobile compatibility (No backend needed!).
- **Luma-Inspired UI**: Dark theme with glassmorphism, blur effects, and smooth animations.
- **Social Sharing**: One-click sharing for LinkedIn, X (Twitter), and Instagram (Download).
- **Responsive Design**: Optimized for iOS Safari and Android Chrome.

## 🛠️ Local Setup

### 1. Installation
```bash
cd frontend
npm install
```

### 2. Run Development Server
```bash
npm run dev
```

---

## 🌐 Deployment (Vercel)

1. Connect your GitHub repository to **Vercel**.
2. Vercel will automatically detect the Vite project.
3. **No environment variables required** (everything is frontend-based).

---

## 📁 Project Structure
- `frontend/src/utils/ticketGenerator.js`: Core Canvas drawing logic.
- `frontend/public/`: Static assets (Template, Background, Logos).
